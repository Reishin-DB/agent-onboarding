# Agent Onboarding — Governance for AI

A FastAPI + React application for AI agent lifecycle governance, auditing, and compliance tracking. Built as a Databricks App.

## Features

- **Agent Registry** — Track all AI agents, their capabilities, and deployment status
- **Governance Workflows** — Approval pipelines for agent deployment
- **Compliance Auditing** — Track data access, model usage, and policy compliance
- **Onboarding Automation** — Guided setup for new agents with UC grants and security policies

## Prerequisites

- Databricks workspace with **Apps** enabled
- Databricks CLI installed and configured
